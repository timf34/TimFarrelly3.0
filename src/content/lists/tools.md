title: "Online tools I recommend"
---

Some tools that I find helpful and use frequently. Items with an \* I use daily. I tell my friends to use items with an !

- [Notion](https://notion.so)*! - for wikis, note taking, and more.
- [diagrams.net](https://app.diagrams.net/) - for building quick flowcharts
- [lucid.app](https://lucid.app/) - more feature rich than app.diagrams
- [hunter.io](https://hunter.io/search) - to help with cold emailing people
- [ClickUp](https://app.clickup.com/)* - for project management
- [Todoist](https://todoist.com)* - as a personal todo list
- [Hacker News](https://news.ycombinator.com/news)* - for procrastination and spiritual enlightenment
- [Jetbrains IDE's](https://www.jetbrains.com/)* - favourite IDEs (free for students!)
- [Built With](https://builtwith.com/)* - for insights into the tech stack of sites I use
- [GitHub](https://github.com/)* - for hosting repos + hosting sites via GitHub Pages
- [ChatGPT](https://chat.openai.com/chat)*! - AI AMA mentor
- [GitHub Copilot](https://github.com/features/copilot)*! - AI powered code completion
- [Connected Papers](https://www.connectedpapers.com/)! - best tool for finding relevant research papers
